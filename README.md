# Python for Data Analysts

## Phase 1: The Python Foundation (The "Toolbox")

Before we use the power tools, we need to know how to use a screwdriver. We will cover just enough core Python to control our data workflow.

- **Step 1: The Environment** – Setting up Jupyter Notebooks (our digital laboratory).
- **Step 2: Variables & Data Types** – Storing numbers, text (strings), and logic (booleans).
- **Step 3: Lists & Dictionaries** – Organizing data into collections (crucial for datasets).
- **Step 4: Logic & Loops** – How to make decisions (if/else) and repeat tasks automatically (for loops).
- **Step 5: Functions** – creating reusable mini-programs to save time.

## Phase 2: Data Manipulation with Pandas (The "Workbench")

This is where 80% of a Data Analyst's life is spent. Pandas is the Excel of Python, but 100x faster and more powerful.

- **Step 6: The DataFrame** – Understanding rows, columns, and the Index.
- **Step 7: Reading Data** – Loading CSVs, Excel files, and JSON.
- **Step 8: Inspection & Cleaning** – Handling missing values (NaN), duplicates, and weird formatting.
- **Step 9: Slicing & Dicing** – Filtering data (e.g., "Show me sales only from 2023").
- **Step 10: Aggregation** – The groupby function (Pivot Tables in Python) to calculate sums, averages, and counts.

## Phase 3: Visualization with Matplotlib (The "Canvas")

Numbers are good; pictures are better. We will learn to tell stories with charts.

- **Step 11: The Anatomy of a Plot** – X-axis, Y-axis, titles, and legends.
- **Step 12: Core Charts** – Line charts (trends), Bar charts (comparison), and Scatter plots (correlation).
- **Step 13: Styling** – Changing colors, sizes, and fonts to make professional reports.

## Phase 4: The Capstone Project

We will take a real-world messy dataset (e.g., Retail Sales or Movie Ratings), clean it, analyze it, and build a final report with charts.
